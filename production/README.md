## 🎯 תיאור המשחק

"הקופה הקטנה פרו" הוא משחק חינוכי מתקדם שמלמד ילדים להעריך מחירים של מוצרים יומיומיים. המשחק כולל ממשק חכם עם סליידר אינטואיטיבי, מצב מורה לכיתות, ומערכת הישגים מתקדמת.

### ✨ תכונות עיקריות

- 🎚️ **סליידר חכם** - ממשק אינטואיטיבי לניחוש מחירים מ-0 עד 10,000₪
- 🎓 **מצב מורה** - ניהול כיתות ומעקב אחר תלמידים
- 🏆 **מערכת הישגים** - עידוד ומוטיבציה לילדים
- 🌙 **מצב לילה/יום** - נוחות עיניים לכל שעה
- 📱 **רספונסיבי** - פועל מושלם על כל המכשירים
- 🧾 **מערכת קבלות** - הדפסה ושיתוף תוצאות
- ⌨️ **קיצורי מקלדת** - נגישות מלאה
- 💾 **שמירה מקומית** - שמירת התקדמות וההעדפות

## 🎮 רמות קושי

### 📱 רמה קלה (עד 200₪)
מוצרי מכולת, חטיפים, משקאות ופריטים יומיומיים

### 🎯 רמה בינונית (עד 3,000₪)  
צעצועים, ספרים, בגדים ומוצרי חשמל בסיסיים

### 🔥 רמה קשה (עד 10,000₪)
מחשבים, מוצרי חשמל מתקדמים וריהוט

## 🏆 מערכת ניקוד

| דיוק הניחוש | נקודות | תיאור |
|-------------|---------|--------|
| מדויק (0%) | 15 | 🎯 ניחוש מושלם! |
| מעולה (עד 5%) | 12 | 🔥 מדהים! |
| טוב (עד 10%) | 10 | 👍 נהדר! |
| בינוני (עד 30%) | 6 | 📈 לא רע |
| נמוך (עד 50%) | 3 | 📚 לומדים |
| רחוק (50%+) | 1 | 🎯 נסה שוב |

**בונוס רצף:** +1 נקודה על כל 3 ניחושים רצופים (מקסימום +5)

## ⌨️ קיצורי מקלדת

- **Enter/Space** - אשר ניחוש
- **H** - הצג רמז (עם קנס של 2 נקודות)
- **S** - דלג על שאלה
- **Escape** - סגור חלונות
- **מקשי מספרים** - בחירה מהירה (בתכנון)

## 🎨 התאמה אישית

### הוספת מוצרים חדשים
ערוך את `data.json`:
```json
{
  "id": 21,
  "category": "קטגוריה חדשה",
  "name": "מוצר חדש",
  "icon": "🆕",
  "price": 100,
  "minPrice": 50,
  "maxPrice": 150,
  "difficulty": 2,
  "hint": "רמז על המוצר",
  "image": "assets/images/category/product.jpg"
}
```

### שינוי צבעי נושא
ערוך משתנים ב-`styles.css`:
```css
:root {
  --primary: #2563eb;    /* צבע ראשי */
  --accent: #f59e0b;     /* צבע מבטא */
  --success: #10b981;    /* צבע הצלחה */
}
```
