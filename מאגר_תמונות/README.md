## 🖼️ דרישות תמונות

### איכות ורזולוציה
- **רזולוציה מינימלית**: 800x600 פיקסלים
- **רזולוציה מומלצת**: 1200x900 פיקסלים
- **פורמט**: JPG או PNG
- **איכות**: גבוהה (80% לפחות)

### תוכן והצגה
- תמונות ברורות וחדות של המוצר
- רקע נייטרלי או מינימליסטי
- תאורה טובה וצבעים נאמנים
- המוצר יהיה במרכז התמונה
- ללא טקסט או לוגואים מסחריים

### מקורות מומלצים
1. **Unsplash** - תמונות ברישיון חופשי
2. **Pexels** - תמונות איכותיות ללא תשלום
3. **Pixabay** - מגוון רחב של תמונות
4. **צילום עצמי** - לייחודיות מקסימלית

## 🔗 כיצד להוסיף תמונות

1. **הורד תמונה איכותית** מאחד המקורות המומלצים
2. **שנה את שם הקובץ** לשם המוצר בעברית (ללא רווחים)
3. **שמור בתיקיה המתאימה** לפי הקטגוריה
4. **עדכן את קובץ המשחק** עם הנתיב החדש

## 🎨 טיפים לבחירת תמונות

1. **בחר תמונות עם רקע נקי** - כדי שהמוצר יבלט
2. **וודא שהצבעים נאמנים** - חשוב לזיהוי נכון
3. **בחר זוויות צילום אופייניות** - כפי שרואים במציאות
4. **הימנע מתמונות עם מחירים** - כדי לא לחשוף את התשובה
5. **בדוק איכות ובהירות** - גם במכשירים ניידים
