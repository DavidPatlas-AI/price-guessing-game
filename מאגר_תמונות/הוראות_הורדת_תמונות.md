# 📥 הוראות הורדת תמונות למשחק

## 🎯 מדריך מהיר להורדת תמונות איכותיות

### שלב 1: גישה לאתרי תמונות חופשיות

#### Unsplash (המומלץ ביותר) 🏆
1. כנס לאתר: https://unsplash.com
2. חפש באנגלית: "computer mouse", "apple fruit", "smartphone" וכו'
3. בחר תמונה איכותית עם רקע נקי
4. לחץ על "Download free" בגודל הגדול ביותר

#### Pexels 📸
1. כנס לאתר: https://www.pexels.com
2. חפש במילות מפתח באנגלית
3. בחר תמונה ברזולוציה גבוהה
4. לחץ על "Free Download"

#### Pixabay 🖼️
1. כנס לאתר: https://pixabay.com
2. השתמש בחיפוש באנגלית
3. בחר "Photos" ו"High Resolution"
4. הורד בגודל המקסימלי

### שלב 2: מילות מפתח לחיפוש

#### טכנולוגיה
- `computer mouse wireless` - עכבר מחשב
- `mechanical keyboard gaming` - מקלדת מכנית
- `modern laptop computer` - מחשב נייד
- `smartphone mobile phone` - סמארטפון

#### מוצרי חשמל
- `stainless steel refrigerator` - מקרר נירוסטה
- `flat screen television` - טלוויזיה שטוחה
- `microwave oven kitchen` - מיקרוגל
- `electric kettle modern` - קומקום חשמלי

#### מזון
- `fresh red apples` - תפוחים אדומים
- `milk bottle glass` - בקבוק חלב
- `sliced bread loaf` - לחם פרוס

#### צעצועים

### שלב 3: בחירת תמונה איכותית

#### מה לחפש ✅
- רקע נקי ומינימליסטי
- המוצר במרכז התמונה
- תאורה טובה וצבעים בהירים
- רזולוציה של לפחות 1200x800
- ללא טקסט או לוגואים

#### מה להימנע ❌
- תמונות עם מחירים או תווים מסחריים
- רקע עמוס או מבלבל
- תמונות מטושטשות או חשוכות
- גדלים קטנים מדי
- תמונות עם זכויות יוצרים

### שלב 4: הורדה ושמירה

1. **הורד את התמונה** בגודל הגדול ביותר (1920x1080 או יותר)
2. **שנה את שם הקובץ** לשם המוצר בעברית:
   - `עכבר_מחשב.jpg`
   - `תפוחים_אדומים.jpg`
   - `לגו_קטן.jpg`
3. **שמור בתיקיה המתאימה** לפי הקטגוריה
4. **בדוק את איכות התמונה** במסך המשחק

### שלב 5: אופטימיזציה

#### דחיסת תמונות (אופציונלי)
- השתמש בכלים כמו TinyPNG או JPEG-Optimizer
- שמור איכות של 80-90%
- הקטן לגודל מקסימלי של 800KB לתמונה

#### שמות קבצים
```
✅ נכון:
- עכבר_מחשב.jpg
- טלוויזיה_55_אינץ.jpg
- שוקולד_מילקה.jpg

❌ לא נכון:
- mouse.jpg
- TV 55".jpg
- chocolate bar.jpg
```

## 🚀 טיפים מתקדמים

### חיפוש מתקדם ב-Unsplash
```
search terms: "minimalist computer mouse white background"
filters: orientation=landscape, color=any
```

### שימוש ב-AI לשיפור תמונות
- Remove.bg - להסרת רקע
- Upscale.media - להגדלת רזולוציה
- Canva - לעריכה בסיסית

### בדיקת איכות
1. פתח את התמונה במסך מלא
2. ודא שהיא חדה ובהירה
3. בדוק שהצבעים נאמנים למציאות
4. וודא שהמוצר מזוהה בקלות
