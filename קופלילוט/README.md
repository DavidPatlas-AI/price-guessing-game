# 🧾 הקופה הקטנה פלוס - משחק ניחוש מחירים לילדים

## 📖 תיאור המשחק

"הקופה הקטנה פלוס" הוא משחק חינוכי אינטראקטיבי שמלמד ילדים להעריך מחירים ולחשב סטיות באחוזים. המשחק מדמה חוויית קופה רושמת אמיתית עם מקלדת, תצוגת LCD, ומערכת ניקוד מתקדמת.

## 🎯 מטרות חינוכיות

- **הערכת מחירים**: למידה על עלויות מוצרים יומיומיים
- **חשיבה כמותית**: הבנת יחסים בין מספרים
- **חישוב אחוזים**: הבנת סטיות ודיוק
- **קבלת החלטות**: פיתוח שיקול דעת כלכלי

## 🛠️ הקמת המשחק

### התקנה מקומית
1. הורד את כל הקבצים לתיקייה אחת
2. פתח את `index.html` בדפדפן
3. התחל לשחק!

### שרת מקומי (אופציונלי)
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

## 🎲 איך לשחק

1. **בחר רמת קושי** מהתפריט הנפתח
2. **הסתכל על המוצר** - שם, אייקון וקטגוריה
3. **הקלד את הניחוש שלך** עם כפתורי המקלדת
4. **לחץ על התצוגה** או כפתור Enter לאישור
5. **קבל נקודות** לפי דיוק הניחוש
6. **עקב אחר ההתקדמות** בטאב "קבלה"

## ⌨️ קיצורי מקלדת

- **מספרים (0-9)**: הקלדת מחיר
- **Enter/רווח**: אישור ניחוש
- **Escape/Delete**: ניקוי הקלט
- **Ctrl+R**: איפוס המשחק

## 🐛 פתרון בעיות נפוצות

### אין צליל
- וודא שהדפדפן לא במצב שקט
- בדוק שקבצי הצליל קיימים בתיקיית `sounds/`
- נסה לטעון מחדש את הדף

### לא שומר נתונים
- וודא שהדפדפן תומך ב-LocalStorage
- בדוק שלא במצב פרטי/אנונימי
- נקה את זיכרון הדפדפן ונסה שוב

### עיצוב לא נטען
- וודא שקובץ `styles.css` באותה תיקייה
- בדוק בקונסולת הדפדפן לשגיאות
- נסה לרענן עם Ctrl+F5
